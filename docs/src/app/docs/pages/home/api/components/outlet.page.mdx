# Outlet

The `Outlet` React component is a special component that is used to render the content of the current page. It is used in the `Layout` component to render the content of the current page.

## Usage 

<Tabs tabs={[{ title: 'TypeScript' }, { title: 'JavaScript' }]} activeIndex={0}>
  <Tabs.Item>
    ```tsx title="app.layout.tsx" {10} showLineNumbers copy /Outlet/
    import { LayoutComponent, Outlet } from "rasengan";
    
    const AppLayout: LayoutComponent = () => {
      return (
        <div>
          <header>
            <h1>Header</h1>
          </header>
          <main>
            <Outlet />
          </main>
        </div>
      );
    };

    AppLayout.path = "/";

    export default AppLayout;
    ```
  </Tabs.Item>
  <Tabs.Item>
