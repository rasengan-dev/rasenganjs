# Router Component

`RouterComponent` is a class provided by Rasengan.js used to create Routers in your application.

{/* <AlertBox type="tip">
  Learn what [`Routers`](/docs/core/routing/base-concepts#router) are here.
</AlertBox> */}

## Usage

<Tabs tabs={[{ title: 'TypeScript' }, { title: 'JavaScript' }]} activeIndex={0}>
  <Tabs.Item>
    ```tsx title="app.router.tsx" showLineNumbers /RouterComponent/
    import { RouterComponent, defineRouter } from "rasengan";

    class AppRouter extends RouterComponent {};

    export default defineRouter({
      imports: [], // Import others routers
      layout: null, // Set a layout
      pages: [] // Import pages
    })(AppRouter)
    ```
  </Tabs.Item>
  <Tabs.Item>

    ```jsx title="app.router.jsx" showLineNumbers /RouterComponent/
      layout: null, // Set a layout
      pages: [] // Import pages
    })(AppRouter)
    ```
  </Tabs.Item>
</Tabs>

## Properties

Here is the list of all the properties of `RouterComponent` class.

| Properties          | Type                                                                 | Description                                        |
| ------------------- | -------------------------------------------------------------------- | -------------------------------------------------- |
| `routers`           | `RouterComponent[]`                                                  | An array of other `Router` classes to be imported. |
| `layout`            | [`LayoutComponent`](/docs/api-reference/components/layout-component) | The layout component to be used.                   |
| `pages`             | [`PageComponent[]`](/docs/api-reference/components/page-component)   | An array of page components to be used.            |
| `notFoundComponent` | `React.FC`                                                           | The 404 page component to be used.                 |
| `loaderComponent`   | `React.FC`                                                           | The loader component to be used.                   |
| `useParentLayout`   | `boolean`                                                            | Whether to use the parent layout or not.           |

Rather than setting these values manually, the [`defineRouter`](/docs/api-reference/functions/define-router) does it for us and create a new instance of `RouterComponent` class and return it.
