# Link Component

The `<Link>` component is a React component that allows you to navigate between routes in your application.
It is the recommended way to navigate between routes in Rasengan.js.

## Usage

### to Property

Here's an example of how to use the `<Link>` component:

<Tabs tabs={[{ title: 'TypeScript' }, { title: 'JavaScript' }]} activeIndex={0}>
  <Tabs.Item>
    ```tsx title="src/app/home.page.tsx" showLineNumbers
    import React from "react";
    import { PageComponent, Link } from "rasengan";

    const Home: PageComponent = () => {
      return (
        <div>
          <h1>Home</h1>
          <Link to="/dashboard">Go to Dashboard</Link>
        </div>
      );
    };

    Home.path = "/";
    Home.metadata = {
      title: "Home",
      description: "Home Page",
    };

    export default Home;
    ```
  </Tabs.Item>
  <Tabs.Item>

Then, you can access to the `state` like follow:

```js title="JS" showLineNumbers copy /useLocation()/
const { state } = useLocation();
```

## Properties

| Props   | Type     | Description                               |
| ------- | -------- | ----------------------------------------- |
| `to`    | `string` | The path to navigate to                   |
| `state` | `any`    | Data sent via URL to the destination page |
