# Overview

**Chidori** is a modern documentation framework built on top of **Rasengan.js**, designed to help you create fast, elegant, and developer-friendly documentation websites.

It focuses on simplicity, performance, and a great developer experience so you can spend less time configuring and more time writing.

## Why Chidori?

- ⚡ Lightning-fast performance
- 🧩 MDX support (Markdown + React components)
- 🎨 Fully customizable
- 🧭 File-based routing
- 🔍 Built-in search (optional)
