# Introduction

Chidori is part of the **Rasengan ecosystem**, a collection of tools designed to improve the way developers build modern applications.

While **Rasengan.js** focuses on building applications, **Chidori** focuses on documenting them.

## Who is it for?

- Framework creators
- Open-source maintainers
- Developer teams
- Anyone who wants clean and powerful documentation

## Philosophy

Chidori is built around three principles:

- **Speed** → instant feedback and fast navigation
- **Clarity** → simple structure and readable content
- **Power** → flexibility when you need it
